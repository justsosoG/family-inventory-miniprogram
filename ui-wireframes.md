# 界面原型描述

## 1. 设计规范

### 1.1 色彩方案

| 颜色 | 色值 | 用途 |
|-----|------|-----|
| 主色 | #07C160 | 微信绿，品牌色 |
| 辅色 | #1890FF | 操作按钮 |
| 警告色 | #FF9500 | 即将过期 |
| 危险色 | #FF4D4F | 已过期 |
| 成功色 | #52C41A | 正常状态 |
| 文字主色 | #333333 | 主要文字 |
| 文字辅色 | #666666 | 次要文字 |
| 文字提示色 | #999999 | 提示文字 |
| 背景色 | #F5F5F5 | 页面背景 |
| 卡片背景 | #FFFFFF | 卡片背景 |

### 1.2 字体规范

| 场景 | 字号 | 字重 |
|-----|------|-----|
| 页面标题 | 18px | Medium |
| 卡片标题 | 16px | Medium |

| 辅助文字 | 12px | Regular |
| 大字体模式 | +4px | - |

### 1.3 间距规范

| 类型 | 数值 |
|-----|------|
| 页面边距 | 16px |

| 大间距 | 16px |
| 小间距 | 4px |

### 1.4 圆角规范

| 元素 | 圆角 |
|-----|------|
| 卡片 | 8px |

## 4. 交互说明

### 4.1 添加物品流程

```
点击添加按钮
    ↓
进入添加页面
    ↓
填写物品信息（支持语音输入）
    ↓
选择分类和位置
    ↓
设置过期时间
    ↓
点击保存
    ↓
保存成功，返回列表
```

### 4.2 处理过期提醒流程

```
收到推送消息
    ↓
点击进入提醒页面
    ↓
查看过期物品详情
    ↓
选择处理方式：
  - 已消耗 → 记录消耗，物品数量 -1
  - 已丢弃 → 删除物品
  - 延期 → 修改过期时间
    ↓
处理完成，提醒标记为已处理
```

### 4.3 搜索物品流程

```
点击搜索框
    ↓
输入关键词
    ↓
实时显示搜索结果
    ↓
点击物品查看详情
```

## 6. 适配说明

### 6.1 大字体模式
- 所有文字字号 +4px
- 卡片高度自适应
- 按钮高度增加

### 6.2 深色模式
- 背景色：#1A1A1A
- 卡片背景：#2D2D2D
- 文字颜色：#FFFFFF / #CCCCCC

### 6.3 屏幕适配
- 最小支持：iPhone SE (375x667)
- 最大支持：iPad (1024x768)
- 全面屏适配：安全区域处理
